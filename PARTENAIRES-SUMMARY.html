✏️ 3 minutes pour personnaliser

Pour chaque partenaire, modifiez juste 4 éléments :

  1. L'URL : href="https://exemple.com"
  2. Le nom : "Partenaire 1" → Votre nom
  3. La description : Expliquez le partenariat
  4. L'icône/logo : Font Awesome ou image

🧪 Tests recommandés

  2. Vérifiez que les 3 cartes s'affichent bien
  3. Cliquez sur chaque partenaire → doit ouvrir dans nouvel onglet
  4. Testez le bouton "Contactez-nous" → redirige vers #contact
  5. Testez sur mobile (F12 → Toggle device toolbar)

🎨 Personnaliser le design

Vous pouvez aussi modifier les couleurs :

  • text-red-600 → couleur des icônes (rouge)
  • bg-red-600 → couleur du bouton (rouge)
  • hover:border-red-600 → bordure au survol
  • to-white → couleur de fond dégradé

Remplacez les codes couleur selon vos préférences (utilisez Tailwind CSS).
