📑 RÉSUMÉ

🚀 Intégration en 3 étapes

1

Ajouter le CSS dans <head>

Une seule ligne à ajouter dans la section HEAD de index.html juste avant </head>

<link rel="stylesheet" href="cookies-consent.css">
2

Ajouter le JavaScript avant </body>

Une seule ligne à ajouter avant la fermeture du body

<script defer src="cookies-consent.js"></script>
3

Tester dans le navigateur

Ouvrez votre site et la bannière apparaît automatiquement au premier chargement

F5 pour recharger → 🍪 La bannière s'affiche!
