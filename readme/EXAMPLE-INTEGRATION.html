🔒 Exemple d'Intégration des Cookies dans index.html

Ce fichier montre AVANT et APRÈS où ajouter les deux imports pour le système de gestion des cookies.

⚠️ Important : Vous n'avez que 2 lignes à ajouter à votre fichier existant!

1️⃣ Ajouter le CSS dans <head>

❌ AVANT

Votre <head> ressemble à ça:

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meca Auto30...</title> <script src="https://cdn.tailwindcss.com"></script> <link rel="stylesheet" href="..."> <style> ...vos styles CSS... </style> </head>

✅ APRÈS

Ajoutez cette ligne juste avant la fermeture </head>:

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meca Auto30...</title> <script src="https://cdn.tailwindcss.com"></script> <link rel="stylesheet" href="..."> <style> ...vos styles CSS... </style> <!-- Système de gestion des cookies RGPD --> <link rel="stylesheet" href="cookies-consent.css"> </head>
ℹ️ Astuce: Cette ligne doit être juste avant </head> pour charger les styles de la bannière

2️⃣ Ajouter le JavaScript avant </body>

❌ AVANT

Votre <body> se termine comme ça:

<!-- Footer --> <footer> <p>© 2024 Meca Auto30</p> </footer> <!-- Vos scripts --> <script src="app.js"></script> </body>

✅ APRÈS

Ajoutez cette ligne juste avant la fermeture </body>:

<!-- Footer --> <footer> <p>© 2024 Meca Auto30</p> </footer> <!-- Vos scripts --> <script src="app.js"></script> <!-- Initialisation du système de cookies --> <script defer src="cookies-consent.js"></script> </body>
ℹ️ Astuce: L'attribut defer charge le script en arrière-plan et l'exécute après le DOM

📋 Résumé - Les 2 lignes à ajouter

Ligne 1 - Dans <head> :

<link rel="stylesheet" href="cookies-consent.css">

Ligne 2 - Avant </body> :

<script defer src="cookies-consent.js"></script>
✅ C'est tout! Votre système de cookies RGPD est prêt à fonctionner!

⚡ Pourquoi cet ordre?

  1. CSS dans <head>:
    Le navigateur charge les styles avant l'affichage du contenu. Cela évite un "flash" non-stylisé (FOUC).
  2. JS avant </body>:
    Le navigateur charge d'abord tout le DOM, puis exécute le JavaScript. Avec defer, c'est asynchrone et non-bloquant.
  3. Fichiers dans le même répertoire:
    index.html, cookies-consent.css, cookies-consent.js tous au même niveau pour éviter les chemins relatifs complexes.

🎯 Exemple complet d'un index.html modifié

Voici à quoi ressemblerait votre fichier completemennt intégré:

<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Meca Auto30 Saint-Dionisy - Votre Expert Mécanique</title> <script src="https://cdn.tailwindcss.com"></script> <link rel="stylesheet" href="...font awesome..."> <style> /* Vos styles existants */ </style> <!-- AJOUT 1️⃣ : Système de gestion des cookies RGPD --> <link rel="stylesheet" href="cookies-consent.css"> </head> <body> <!-- Votre contenu --> <header>...</header> <main>...</main> <footer>...</footer> <!-- Vos scripts existants --> <script src="app.js"></script> <!-- AJOUT 2️⃣ : Initialisation du système de cookies --> <script defer src="cookies-consent.js"></script> </body> </html>

✅ Après l'intégration - Vérifications

1. Vérifier que tout est en place

  • ☑️ cookies-consent.css est dans le répertoire racine
  • ☑️ cookies-consent.js est dans le répertoire racine
  • ☑️ Les 2 lignes d'import sont dans index.html

2. Tester dans le navigateur

  • ☑️ Ouvrez votre site (F5 pour forcer le rechargement)
  • ☑️ La bannière de cookies doit apparaître en bas 🍪
  • ☑️ Testez les 3 boutons (Accepter, Refuser, Personnaliser)

3. Vérifier la sauvegarde (DevTools)

  • ☑️ Appuyez sur F12 (DevTools)
  • ☑️ Allez dans "Application" → "Local Storage"
  • ☑️ Vous devriez voir meca-auto-cookies-consent
  • ☑️ Les préférences doivent être sauvegardées après chaque action

4. Tester sur mobile

  • ☑️ F12 → Toggle device toolbar (ou Ctrl+Shift+M)
  • ☑️ La bannière doit être responsive
  • ☑️ Les boutons doivent être empilés verticalement

🐛 Troubleshooting courant

❌ Je ne vois pas la bannière

Solution:
  1. Vérifiez F12 → Network que cookies-consent.css et cookies-consent.js se chargent
  2. Vérifiez F12 → Console pour les erreurs en rouge
  3. Si le JS ne charge pas, vérifiez le chemin du fichier
  4. Rechargez la page complètement: Ctrl+Shift+R ou Cmd+Shift+R

❌ La bannière s'affiche mais n'est pas stylisée

Solution:
  1. Vérifiez que le CSS est chargé (F12 → Network)
  2. Peut-être un conflit avec Tailwind CSS
  3. Vérifiez que le chemin est correct: href="cookies-consent.css"

❌ La bannière s'affiche à chaque rechargement

Solution:
  1. Vérifiez F12 → Application → Local Storage
  2. Vous devriez voir meca-auto-cookies-consent
  3. Si ce n'est pas là, le JS ne s'exécute pas
  4. Vérifiez F12 → Console pour les erreurs

❌ Les imports HTML "404 Not Found"

Solution:
  1. Assurez-vous que cookies-consent.css et cookies-consent.js sont dans le même répertoire que index.html
  2. Vérifiez les noms: pas d'espaces, pas d'accents, respect de la casse
  3. Si dans un sous-dossier: modifiez le chemin: href="assets/cookies-consent.css"

🚀 Prochaines étapes recommandées

  1. Mettre à jour votre politique de confidentialité
    Ajoutez une section "Cookies" dans politique-confidentialite.html expliquant quels cookies vous utilisez et pourquoi.
  2. Implémenter le chargement conditionnel des analytics
    Si vous utilisez Google Analytics, ne chargez le script QUE si l'utilisateur a accepté les analytics.
  3. Tester la conformité RGPD
    Assurez-vous que aucun service externe n'est chargé avant le consentement explicite.
  4. Supprimer les fichiers d'exemple
    Une fois intégré, vous pouvez supprimer:
    • COOKIES-INTEGRATION.html
    • EXAMPLE-INTEGRATION.html (ce fichier)
