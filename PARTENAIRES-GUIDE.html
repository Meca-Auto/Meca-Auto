🤝 Personnaliser la Section Partenaires

Guide complet pour remplacer les partenaires par les vôtres

📍 Où se trouve la section?

La section "Partenaires" (ou "Collaborations") a été ajoutée dans votre index.html entre la section Témoignages et la section Contact.

✅ Elle est aussi accessible via le menu : Accueil → Partenaires

✏️ Étape 1 : Modifier les 3 partenaires

Chaque partenaire a cette structure dans le code :

<!-- Partner 1 --> <a href="https://example1.com" target="_blank" class="group"> <div class="bg-white rounded-xl p-8 shadow-lg hover:shadow-2xl..."> <!-- ICÔNE --> <div class="mb-6 h-20 flex items-center justify-center"> <i class="fas fa-car text-5xl text-red-600"></i> </div> <!-- NOM --> <h3 class="text-xl font-bold...">Partenaire 1</h3> <!-- DESCRIPTION --> <p class="text-gray-600...">Description du partenariat</p> <!-- BOUTON --> <div class="inline-block bg-red-600..."> Visiter le site → </div> </div> </a>

Comment personnaliser ?

  1. Changez l'URL : https://example1.com → votre URL réelle
  2. Changez le nom : "Partenaire 1" → le vrai nom
  3. Changez la description : "Description du partenariat" → votre texte
  4. Changez l'icône : Voir section ci-dessous

🎨 Étape 2 : Changer l'icône

Par défaut, j'ai utilisé des icônes Font Awesome. Vous avez 3 options :

Option A : Changer l'icône Font Awesome (Plus simple)

Icônes disponibles :

  • <i class="fas fa-car"></i> - Voiture 🚗
  • <i class="fas fa-tools"></i> - Outils 🔧
  • <i class="fas fa-cog"></i> - Engrenage ⚙️
  • <i class="fas fa-wrench"></i> - Clé anglaise 🔨
  • <i class="fas fa-industry"></i> - Usine 🏭
  • <i class="fas fa-truck"></i> - Camion 🚚

Pour en trouver d'autres : FontAwesome Icons

Option B : Ajouter votre propre logo (si c'est une image)

Remplacez cette ligne :

<i class="fas fa-car text-5xl text-red-600"></i>

Par :

<img src="logo-partenaire1.png" alt="Nom Partenaire" style="max-height: 80px; max-width: 120px;">

Et mettez le fichier logo dans le même dossier que votre index.html

Option C : Ajouter le nom du partenaire à la place de l'icône

Remplacez :

<i class="fas fa-car text-5xl text-red-600"></i>

Par :

<div class="text-3xl font-bold text-red-600">PARTENAIRE 1</div>

📝 Exemple de personnalisation complète

Voici comment personnaliser un partenaire avec un vrai logo :

<!-- Partenaire: Bosch --> <a href="https://www.bosch.com/fr/" target="_blank" class="group"> <div class="bg-white rounded-xl p-8 shadow-lg hover:shadow-2xl..."> <div class="mb-6 h-20 flex items-center justify-center"> <img src="logo-bosch.png" alt="Bosch" style="max-height: 80px;"> </div> <h3 class="text-xl font-bold...">Bosch</h3> <p class="text-gray-600...">Pièces et outils de diagnostic automobile de qualité professionnelle</p> <div class="inline-block bg-red-600..."> Visiter le site → </div> </div> </a>
✅ Remplacez "Bosch" et l'URL par vos vrais partenaires !

🔗 Étape 3 : Personnaliser l'appel à l'action

Le texte en bas "Intéressé par une collaboration ?" peut aussi être modifié :

<h3 class="text-2xl md:text-3xl font-bold mb-4">Intéressé par une collaboration ?</h3> <p class="text-lg mb-8 text-red-100"> Nous aimerions explorer les possibilités de partenariat avec votre entreprise. </p>

Vous pouvez personnaliser :

  • Le titre principal ("Intéressé par une collaboration ?")
  • La description ("Nous aimerions explorer...")
  • Le texte du bouton ("Contactez-nous")

📋 Template à copier

Voici un template vide à copier/coller pour ajouter une 4ème partenaire si besoin :

<!-- Partner 4 --> <a href="https://votre-url.com" target="_blank" class="group"> <div class="bg-white rounded-xl p-8 shadow-lg hover:shadow-2xl transition-all duration-300 hover:-translate-y-2 text-center cursor-pointer border border-gray-200 hover:border-red-600"> <div class="mb-6 h-20 flex items-center justify-center"> <i class="fas fa-star text-5xl text-red-600"></i> </div> <h3 class="text-xl font-bold text-gray-800 mb-2 group-hover:text-red-600 transition">Votre Partenaire</h3> <p class="text-gray-600 text-sm mb-4">Description courte</p> <div class="inline-block bg-red-600 text-white px-4 py-2 rounded-full text-sm font-semibold group-hover:bg-red-700 transition"> Visiter le site → </div> </div> </a>
ℹ️ À faire :
  1. Remplacez "https://votre-url.com"
  2. Remplacez "Votre Partenaire"
  3. Remplacez "Description courte"
  4. Changez l'icône ou le logo

✅ Points importants

Assurez-vous que :
  • ✅ Les URLs sont correctes (commencent par https://)
  • ✅ Les noms des partenaires sont exacts
  • ✅ Les descriptions sont pertinentes
  • ✅ Les icônes ou logos s'affichent bien
  • ✅ Les liens functionnent (testez-les en cliquant)

🧪 Test

  1. Modifiez votre index.html
  2. Ouvrez votre site : F5 pour recharger
  3. Allez à la section "Partenaires" via le menu
  4. Les partenaires s'affichent avec votre contenu ✅
  5. Cliquez sur les logos pour vérifier les liens
